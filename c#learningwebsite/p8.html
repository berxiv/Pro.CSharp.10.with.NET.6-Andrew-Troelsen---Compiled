Chapter 8: Working with Interfaces

What is an Interface?

Interface: A way to define a set of methods and properties that a class must implement.

Purpose: To allow different classes to share the same set of methods and properties, even if they are not related.

public interface IExample {
    void DoSomething();
    int GetNumber();
}

Why Use Interfaces?

Multiple Behaviors: A class can implement multiple interfaces, allowing it to have multiple behaviors.

Flexibility: Interfaces allow you to define methods that can be used by different classes in different ways.

public class MyClass : IExample, IAnotherExample {
    public void DoSomething() {
        // Implementation
    }

    public int GetNumber() {
        return 42;
    }

    public void AnotherMethod() {
        // Another implementation
    }
}

Implementing Interfaces

Syntax: Use the : InterfaceName syntax after the class name.

All or Nothing: You must implement all methods and properties defined in the interface.

public class MyClass : IExample {
    public void DoSomething() {
        Console.WriteLine("Doing something!");
    }

    public int GetNumber() {
        return 42;
    }
}

Default Methods in Interfaces (C# 8.0)

Default Methods: Interfaces can have methods with default implementations.

Optional Override: Classes can override these methods if needed.

public interface IExample {
    void DoSomething();
    int GetNumber() => 42; // Default implementation
}

Exercises

Exercise 1: Create and Implement an Interface

Task: Create an interface IMovable with methods Move() and Stop().

Implement: Create a class Car that implements IMovable.

interface IMovable {
    method Move()
    method Stop()
}

class Car implements IMovable {
    method Move() {
        print "Car is moving"
    }

    method Stop() {
        print "Car has stopped"
    }
}

Exercise 2: Use Default Methods

Task: Add a default method GetSpeed() to IMovable that returns a default speed.

Implement: Override GetSpeed() in the Car class to return a specific speed.

interface IMovable {
    method Move()
    method Stop()
    method GetSpeed() {
        return 60 // Default speed
    }
}

class Car implements IMovable {
    method Move() {
        print "Car is moving"
    }

    method Stop() {
        print "Car has stopped"
    }

    method GetSpeed() {
        return 100 // Car's speed
    }
}

Exercise 3: Check Interface Implementation

Task: Create a method CheckMovable that takes an IMovable object and calls its methods.

Implement: Create instances of Car and check if they implement IMovable.

method CheckMovable(IMovable obj) {
    obj.Move()
    print obj.GetSpeed()
    obj.Stop()
}

Car myCar = new Car()
CheckMovable(myCar)
