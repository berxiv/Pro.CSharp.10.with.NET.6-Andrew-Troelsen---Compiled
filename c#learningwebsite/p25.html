Introducing Windows Presentation Foundation (WPF) and XAML

What is WPF?

  • WPF is a tool for creating desktop applications with graphical user interfaces (GUIs).
  • It was introduced with .NET 3.0 and is an alternative to older tools like Windows Forms and GDI+.
  • WPF allows you to create applications with 2D and 3D graphics, animations, and more, all in one place.

Key Components of WPF

  • Application: The main class that runs your WPF application.
  • Window: Represents a window in your application.
  • ContentControl: A control that can hold a single piece of content.
  • Control: The base class for all WPF controls like buttons and text boxes.
  • UIElement: Provides basic functionality like handling input (mouse, keyboard).
  • FrameworkElement: Adds more features like data binding and layout.
  • Visual: Handles rendering (drawing) of graphics.
  • DependencyObject: Supports special properties called dependency properties.
  • DispatcherObject: Manages threading and concurrency.

What is XAML?

  • XAML (Extensible Application Markup Language) is a way to define the UI of a WPF application using XML.
  • It separates the design (look and feel) from the code (logic).

Benefits of WPF and XAML

  • Unified API: Everything you need for GUI development is in one place.
  • Separation of Concerns: Design and logic are separate, making it easier to manage.
  • Rich Graphics: Supports advanced graphics and animations.
  • Data Binding: Easily bind data to UI elements.
  • Customizable Controls: Change the look of controls without writing new code.

Basic WPF Application Structure

  • App.xaml: Defines the application and its startup settings.
  • MainWindow.xaml: Defines the main window of the application.
  • MainWindow.xaml.cs: Contains the code-behind for the main window.

Exercises

Exercise 1: Create a Simple WPF Application

  • Objective: Create a WPF application with a button that shows a message when clicked.
  • Steps:
    1. Create a new WPF project in Visual Studio.
    2. Add a button to the main window.
    3. Write code to show a message when the button is clicked.
  • Pseudocode: 
    
Create a new WPF project
Add a button to the main window
Handle the button's click event
Show a message box when the button is clicked

Exercise 2: Add a TextBox and Display Text

  • Objective: Add a TextBox and a Button. When the button is clicked, display the text from the TextBox in a MessageBox.
  • Steps:
    1. Add a TextBox and a Button to the main window.
    2. Handle the button's click event.
    3. Get the text from the TextBox and display it in a MessageBox.
  • Pseudocode: 
    
Add a TextBox and a Button to the main window
Handle the button's click event
Get the text from the TextBox
Show the text in a MessageBox

Exercise 3: Change Button Color on Click

  • Objective: Change the background color of a button when it is clicked.
  • Steps:
    1. Add a Button to the main window.
    2. Handle the button's click event.
    3. Change the button's background color in the event handler.
