Chapter 20: Data Access with ADO.NET

ADO.NET is a set of tools in .NET that helps you work with databases. It includes different types of data providers for different databases like SQL Server, Oracle, and MySQL.

Key Concepts

  • Data Providers: These are specific to each database type and include classes like DbConnection, DbCommand, DbDataReader, and DbDataAdapter.
  • Connection: Used to connect to a database.
  • Command: Used to execute SQL queries.
  • DataReader: Used to read data from the database.
  • DataAdapter: Used to transfer data between the database and your application.

Basic Steps to Access Data

  1. Create a Connection: Connect to the database.
  2. Create a Command: Write your SQL query.
  3. Execute the Command: Run the query and get results.
  4. Read Data: Use a DataReader to read the results.

Example


using (SqlConnection connection = new SqlConnection("your_connection_string"))
{
    connection.Open();
    SqlCommand command = new SqlCommand("SELECT * FROM Inventory", connection);
    SqlDataReader reader = command.ExecuteReader();
    while (reader.Read())
    {
        Console.WriteLine(reader["ColumnName"]);
    }
}

Pseudocode:


1. Create a connection string.
2. Open a connection.
3. Create a command with a SELECT query.
4. Execute the command.
5. Read and print the results.

using (SqlConnection connection = new SqlConnection("your_connection_string"))

Exercise 3: Insert a Record

Goal: Write a program to insert a new record into a table.

Pseudocode:


1. Create a connection string.
2. Open a connection.
3. Create a command with an INSERT query.
4. Execute the command.
5. Print "Record inserted".
6. Close the connection.

Hint:


using (SqlConnection connection = new SqlConnection("your_connection_string"))
{
    connection.Open();
    SqlCommand command = new SqlCommand("INSERT INTO Inventory (Column1, Column2) VALUES (Value1, Value2)", connection);
    command.ExecuteNonQuery();
    Console.WriteLine("Record inserted");
    connection.Close();
}

Exercise 4: Update a Record

Goal: Write a program to update an existing record in a table.

Pseudocode:

Summary

  • ADO.NET helps you interact with databases.
  • Connections are used to connect to databases.
  • Commands are used to execute SQL queries.
  • DataReaders are used to read data.
  • DataAdapters are used to transfer data between the database and your application.
