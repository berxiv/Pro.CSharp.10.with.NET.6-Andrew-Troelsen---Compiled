Chapter 19: File I/O and Object Serialization

When you create desktop applications, it's common to save information between user sessions. This chapter covers how to handle files and directories in .NET and how to save and load objects.

System.IO Namespace

The System.IO namespace in .NET provides classes for working with files and directories. Here are some key classes:

  • BinaryReader/BinaryWriter: Read and write primitive data types as binary.
  • BufferedStream: Temporary storage for a stream of bytes.
  • Directory/DirectoryInfo: Work with directories.
  • DriveInfo: Information about drives.
  • File/FileInfo: Work with files.
  • FileStream: Random file access.
  • FileSystemWatcher: Monitor changes in files and directories.
  • MemoryStream: Work with data in memory.
  • Path: Work with file and directory paths.
  • StreamWriter/StreamReader: Write and read text files.
  • StringWriter/StringReader: Write and read strings.

Working with Directories

DirectoryInfo: Create, move, delete, and get information about directories.

Directory: Similar to DirectoryInfo but uses static methods.

DirectoryInfo dir = new DirectoryInfo(@"C:\MyFolder");
dir.Create(); // Creates the directory

Working with Files

FileInfo: Create, move, delete, and get information about files.

File: Similar to FileInfo but uses static methods.

FileInfo file = new FileInfo(@"C:\MyFolder\myfile.txt");
file.Create(); // Creates the file

Reading and Writing Files

StreamWriter/StreamReader: For text files.

BinaryWriter/BinaryReader: For binary files.

using (StreamWriter writer = new StreamWriter(@"C:\MyFolder\myfile.txt"))
{
    writer.WriteLine("Hello, World!");
}

Object Serialization

Serialization is saving the state of an object to a file. Deserialization is loading the object back from the file.

XML Serialization: Save objects as XML.

JSON Serialization: Save objects as JSON.

XmlSerializer serializer = new XmlSerializer(typeof(MyClass));
using (FileStream fs = new FileStream(@"C:\MyFolder\myfile.xml", FileMode.Create))
{
    serializer.Serialize(fs, myObject);
}

Exercises

Exercise 1: Create and Write to a File

Task: Create a text file and write "Hello, World!" to it.

using (StreamWriter writer = new StreamWriter("hello.txt"))
{
    writer.WriteLine("Hello, World!");
}

Exercise 2: Read from a File

Task: Read the content of the file created in Exercise 1 and print it to the console.

using (StreamReader reader = new StreamReader("hello.txt"))
{
    string content = reader.ReadToEnd();
    Console.WriteLine(content);
}

Exercise 3: Serialize an Object to XML

Task: Serialize an object of a custom class to an XML file.

MyClass obj = new MyClass();
XmlSerializer serializer = new XmlSerializer(typeof(MyClass));
using (FileStream fs = new FileStream("object.xml", FileMode.Create))
{
    serializer.Serialize(fs, obj);
}

Exercise 4: Deserialize an Object from XML

Task: Deserialize the object from the XML file created in Exercise 3.

XmlSerializer serializer = new XmlSerializer(typeof(MyClass));
using (FileStream fs = new FileStream("object.xml", FileMode.Open))
{
    MyClass obj = (MyClass)serializer.Deserialize(fs);
}
