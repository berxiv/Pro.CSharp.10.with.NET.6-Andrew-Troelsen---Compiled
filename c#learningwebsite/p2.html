Chapter 2: Building C# Applications

Getting Started with .NET 6

To build C# applications, you need to install the .NET 6 SDK and runtime. Here's how:

  1. Go to www.dot.net.
  2. Click "Download" and then "All .NET downloads."
  3. Select ".NET 6.0 (recommended)" and download the SDK for your operating system.

Understanding .NET Version Numbers

Example Version: 6.0.100

  • 6.0: Major version
  • 1: Feature update
  • 00: Patch version

Confirming Installation

Open a command window and type these commands to check your installation:

  • dotnet --version: Shows the SDK version.
  • dotnet --list-runtimes: Lists installed runtimes.
  • dotnet --list-sdks: Lists installed SDKs.

Using Visual Studio 2022

Visual Studio 2022 is an Integrated Development Environment (IDE) for building C# applications. You can download it from Visual Studio Downloads.

Installation Steps

  1. Select the workloads you need (e.g., .NET desktop development, ASP.NET and web development).
  2. Click "Install."

Creating a Simple C# Console Application

  1. Open Visual Studio.
  2. Select "Create a new project."
  3. Choose "Console App" and name it SimpleCSharpConsoleApp.
  4. Write the following code in Program.cs:
Console.Title = "My Rocking App";
Console.ForegroundColor = ConsoleColor.Yellow;
Console.BackgroundColor = ConsoleColor.Blue;
Console.WriteLine("*************************************");
Console.WriteLine("***** Welcome to My Rocking App *****");
Console.WriteLine("*************************************");
Console.BackgroundColor = ConsoleColor.Black;
Console.ReadLine();

Running and Debugging

  • Run: Press Ctrl+F5.
  • Debug: Set breakpoints by clicking the leftmost gray bar in the code editor and press F5.
