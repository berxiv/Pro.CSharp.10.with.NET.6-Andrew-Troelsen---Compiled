Chapter 5: Understanding Encapsulation

What is Encapsulation?

Encapsulation is a way to keep the details of how an object works hidden from the outside world. Think of it like a capsule that holds all the important data and functions inside it.

Creating a Class

A class is like a blueprint for creating objects. It defines the data and the functions that the objects created from the class will have.

class Car {
    public string petName;
    public int currSpeed;
}

Constructors

Constructors are special methods used to initialize objects. They set up the initial state of an object.

class Car {
    public string petName;
    public int currSpeed;

    // Constructor
    public Car() {
        petName = "Chuck";
        currSpeed = 10;
    }
}

Methods

Methods define the behavior of the objects. They are functions that belong to a class.

class Car {
    public string petName;
    public int currSpeed;

    public void PrintState() {
        Console.WriteLine($"{petName} is going {currSpeed} MPH.");
    }

    public void SpeedUp(int delta) {
        currSpeed += delta;
    }
}

Access Modifiers

Access modifiers control the visibility of class members. Common ones are:

  • public: Accessible from anywhere.
  • private: Accessible only within the class.
  • protected: Accessible within the class and its subclasses.

Properties

Properties are a way to control access to the fields of a class. They use get and set methods.

class Car {
    private string petName;
    public string PetName {
        get { return petName; }
        set { petName = value; }
    }
}

Static Members

Static members belong to the class itself rather than any object. They are shared among all instances of the class.

class Car {
    public static int numberOfCars;
}

Exercises

Exercise 1: Create a Simple Class

Create a class named Person with the following properties:

  • Name (string)
  • Age (int)
class Person {
    public string Name;
    public int Age;
}

Exercise 2: Add a Constructor

Add a constructor to the Person class that initializes the Name and Age properties.

class Person {
    public string Name;
    public int Age;

    // Constructor
    public Person(string name, int age) {
        Name = name;
        Age = age;
    }
}

Exercise 3: Create Methods

Add a method to the Person class that prints the person's details.

class Person {
    public string Name;
    public int Age;

    public Person(string name, int age) {
        Name = name;
        Age = age;
    }

    public void PrintDetails() {
        Console.WriteLine($"Name: {Name}, Age: {Age}");
    }
}

Exercise 4: Use Properties

Modify the Person class to use properties for Name and Age.

class Person {
    private string name;
    private int age;

    public string Name {
        get { return name; }
        set { name = value; }
    }

    public int Age {
        get { return age; }
        set { age = value; }
    }

    public Person(string name, int age) {
        Name = name;
        Age = age;
    }

    public void PrintDetails() {
        Console.WriteLine($"Name: {Name}, Age: {Age}");
    }
}

Exercise 5: Add Static Members

Add a static member to the Person class to count the number of Person objects created.

class Person {
    private string name;
    private int age;
    public static int personCount;

    public string Name {
        get { return name; }
        set { name = value; }
    }

    public int Age {
        get { return age; }
        set { age = value; }
    }

    public Person(string name, int age) {
        Name = name;
        Age = age;
        personCount++;
    }

    public void PrintDetails() {
        Console.WriteLine($"Name: {Name}, Age: {Age}");
    }
}
