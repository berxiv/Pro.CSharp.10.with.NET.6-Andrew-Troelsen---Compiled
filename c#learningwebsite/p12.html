Chapter 12: Delegates, Events, and Lambda Expressions

What Are Delegates?

Delegates are like pointers to methods. They allow methods to be passed as parameters. They are type-safe, meaning they ensure the method signature matches the delegate signature. Delegates can point to multiple methods (multicasting).

Example:

public delegate int MathOperation(int x, int y);

This delegate can point to any method that takes two integers and returns an integer.

Creating and Using Delegates

  1. Define a Delegate: 
    public delegate int MathOperation(int x, int y);
  2. Create Methods that match the delegate signature: 
    public static int Add(int x, int y) => x + y;
public static int Subtract(int x, int y) => x - y;
  3. Create Delegate Instances: 
    MathOperation add = new MathOperation(Add);
MathOperation subtract = new MathOperation(Subtract);
  4. Invoke the Delegate: 
    int result = add(10, 20); // Calls Add method

Multicasting with Delegates

You can add multiple methods to a delegate using += and remove them using -=. When invoked, all methods in the delegate's list are called.

Events

Events are a way for a class to notify other classes when something happens. They are based on delegates but provide more control and safety. Use the event keyword to declare an event.

Example:

public delegate void Notify();  // Delegate
public event Notify OnProcessCompleted; // Event

Using Events

  1. Define a Delegate and Event: 
    public delegate void Notify();
public event Notify OnProcessCompleted;
  2. Raise the Event: 
    if (OnProcessCompleted != null)
    OnProcessCompleted();
  3. Subscribe to the Event: 
    obj.OnProcessCompleted += EventHandlerMethod;
  4. Event Handler Method: 
    void EventHandlerMethod()
{
    Console.WriteLine("Process Completed!");
}

Lambda Expressions

Lambda Expressions are a concise way to write anonymous methods. They use the => syntax.

Example:

Func add = (x, y) => x + y;
int result = add(10, 20); // result is 30

Exercises

Exercise 1: Create and Use a Delegate

Task: Create a delegate that points to a method which multiplies two numbers.

Pseudocode:

Define delegate MultiplyDelegate
Create method Multiply that takes two integers and returns their product
Create an instance of MultiplyDelegate pointing to Multiply
Call the delegate with two numbers and print the result

Exercise 2: Implement an Event

Task: Create a class with an event that triggers when a value exceeds a limit.

Pseudocode:

Define delegate LimitExceededHandler
Define event LimitExceeded
Create method CheckValue that checks if a value exceeds a limit
If value exceeds limit, raise LimitExceeded event
Create a method to handle the event and print a message
Subscribe to the event and test with different values

Task: Use a lambda expression to filter even numbers from a list.

Pseudocode:

Create a list of integers
Use a lambda expression with List.FindAll to find even numbers
Print the even numbers
