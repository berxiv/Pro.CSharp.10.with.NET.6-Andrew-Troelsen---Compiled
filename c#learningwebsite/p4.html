Chapter 4: Core C# Programming Constructs, Part 2

Understanding C# Arrays

Arrays are collections of data items of the same type, accessed using a numerical index.

Declaring and Using Arrays

int[] myInts = new int[3]; // Array of 3 integers
string[] booksOnDotNet = new string[100]; // Array of 100 strings

Filling and Accessing Arrays

myInts[0] = 100;
myInts[1] = 200;
myInts[2] = 300;

foreach (int i in myInts) {
    Console.WriteLine(i);
}

Array Initialization Syntax

string[] stringArray = new string[] { "one", "two", "three" };
bool[] boolArray = { false, false, true };
int[] intArray = new int[4] { 20, 22, 23, 0 };

Multidimensional Arrays

Rectangular Arrays: Arrays with rows of the same length.

int[,] myMatrix = new int[3, 4];

Jagged Arrays: Arrays of arrays, where each inner array can have different lengths.

int[][] myJagArray = new int[5][];
for (int i = 0; i < myJagArray.Length; i++) {
    myJagArray[i] = new int[i + 7];
}

Using Arrays as Arguments or Return Values

static void PrintArray(int[] myInts) {
    foreach (int i in myInts) {
        Console.WriteLine(i);
    }
}

static string[] GetStringArray() {
    return new string[] { "Hello", "from", "GetStringArray" };
}

Understanding Methods

Methods are blocks of code that perform a specific task.

Basic Method Structure

static int Add(int x, int y) {
    return x + y;
}

Method Overloading

public static int Add(int x, int y) {
    return x + y;
}

public static double Add(double x, double y) {
    return x + y;
}

Understanding Enumerations

Enumerations (enums) are sets of named constants.

Declaring an Enum

enum EmpTypeEnum {
    Manager,
    Grunt,
    Contractor,
    VicePresident
}

Understanding Structures

Structures (structs) are user-defined types that can contain multiple data fields.

Declaring a Struct

struct Point {
    public int X;
    public int Y;

    public void Display() {
        Console.WriteLine($"X = {X}, Y = {Y}");
    }
}

Nullable Types

Nullable types can represent all the values of their underlying type plus an additional null value.

Declaring Nullable Types

int? nullableInt = 10;
bool? nullableBool = null;

Tuples

Tuples are lightweight data structures that contain multiple fields.

Creating and Using Tuples

(string, int, string) values = ("a", 5, "c");
Console.WriteLine(values.Item1); // "a"
