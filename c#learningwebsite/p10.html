Chapter 10: Collections and Generics

Introduction

When you create applications with .NET Core, you often need to manage data in memory. This data can come from databases, files, web services, or user input. .NET provides two main types of collections to handle this data: nongeneric collections and generic collections.

Why Use Collections?

  • Arrays: Fixed size, can hold items of the same type.
  • Collections: More flexible, can grow and shrink, and can hold specific types of data.

Types of Collections

Nongeneric Collections

Found in System.Collections namespace. They are not type-safe and can hold any type of data.

Generic Collections

Found in System.Collections.Generic namespace. They are type-safe and more efficient.

Common Nongeneric Collections

  • ArrayList: A list that can grow and shrink.
  • Hashtable: Stores key-value pairs.
  • Queue: First-in, first-out collection.
  • Stack: Last-in, first-out collection.

Common Generic Collections

  • List<T>: A list that can grow and shrink, type-safe.
  • Dictionary<TKey, TValue>: Stores key-value pairs, type-safe.
  • Queue<T>: First-in, first-out collection, type-safe.
  • Stack<T>: Last-in, first-out collection, type-safe.

Using Collections

Example: Using List<T>

List<string> names = new List<string>();
names.Add("Alice");
names.Add("Bob");
Console.WriteLine(names.Count); // Output: 2

Exercises

Exercise 1: Create and Use a List

Task: Create a list of integers, add five numbers, and print them.

Pseudocode:

1. Create a list of integers.
2. Add five numbers to the list.
3. Loop through the list and print each number.

Hint:

List<int> numbers = new List<int>();
numbers.Add(1);
numbers.Add(2);
numbers.Add(3);
numbers.Add(4);
numbers.Add(5);
foreach (int number in numbers) {
    Console.WriteLine(number);
}

Exercise 2: Use a Dictionary

Task: Create a dictionary to store names and ages, add three entries, and print them.

Pseudocode:

1. Create a dictionary with string keys and int values.
2. Add three name-age pairs to the dictionary.
3. Loop through the dictionary and print each key-value pair.

Hint:

Dictionary<string, int> people = new Dictionary<string, int>();
people.Add("Alice", 30);
people.Add("Bob", 25);
people.Add("Charlie", 35);
foreach (var person in people) {
    Console.WriteLine($"{person.Key} is {person.Value} years old.");
}

Conclusion

Collections in .NET Core help you manage data efficiently. Generic collections are preferred for their type safety and performance benefits. Practice using different collections to become familiar with their features and capabilities.
