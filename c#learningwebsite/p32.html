Chapter 32: RESTful Services with ASP.NET Core

Introduction

In this chapter, we'll learn how to create RESTful services using ASP.NET Core. RESTful services allow different applications to communicate over the web using standard HTTP methods like GET, POST, PUT, and DELETE.

What is ASP.NET Core?

ASP.NET Core is a framework for building web applications and services. It combines the features of ASP.NET MVC and Web API into a single framework.

Key Concepts

  • HTTP Verbs: Methods used to interact with resources.
    • GET: Retrieve data.
    • POST: Create new data.
    • PUT: Update existing data.
    • DELETE: Remove data.
  • URI (Uniform Resource Identifier): A string that identifies a resource on the web. For example, http://www.example.com/api/cars.
  • JSON (JavaScript Object Notation): A lightweight data format used to exchange data between a client and a server.

Setting Up a RESTful Service

Create a Controller

[Route("api/[controller]")]
[ApiController]
public class ValuesController : ControllerBase
{
    [HttpGet]
    public IActionResult Get()
    {
        return Ok(new string[] { "value1", "value2" });
    }
}

Return JSON Data

[HttpGet]
public IActionResult Get()
{
    return Ok(new string[] { "value1", "value2" });
}

Handle HTTP Status Codes

  • 200 OK: The request was successful.
  • 400 Bad Request: The request was invalid.
  • 404 Not Found: The resource was not found.

Configuring JSON Handling

Camel Case

builder.Services.AddControllers()
    .AddJsonOptions(options =>
    {
        options.JsonSerializerOptions.PropertyNamingPolicy = null;
        options.JsonSerializerOptions.PropertyNameCaseInsensitive = true;
        options.JsonSerializerOptions.WriteIndented = true;
    });

Ignore Reference Cycles

options.JsonSerializerOptions.ReferenceHandler = ReferenceHandler.IgnoreCycles;

Exercises

Exercise 1: Create a Simple API

Objective: Create an API that returns a list of books.

Steps:

  • Create a new ASP.NET Core project.
  • Add a BooksController.
  • Implement a GET method that returns a list of books.

Pseudocode:

[Route("api/[controller]")]
[ApiController]
public class BooksController : ControllerBase
{
    [HttpGet]
    public IActionResult GetBooks()
    {
        var books = new List { "Book1", "Book2", "Book3" };
        return Ok(books);
    }
}

Exercise 2: Handle Different HTTP Verbs

Objective: Implement POST, PUT, and DELETE methods in the BooksController.

Steps:

  • Add methods to create, update, and delete books.
  • Use appropriate HTTP status codes.

Pseudocode:

[HttpPost]
public IActionResult AddBook(string book)
{
    // Add book to the list
    return CreatedAtAction(nameof(GetBooks), new { id = book }, book);
}

[HttpPut("{id}")]
public IActionResult UpdateBook(int id, string book)
{
    // Update book in the list
    return Ok(book);
}

[HttpDelete("{id}")]
public IActionResult DeleteBook(int id)
{
    // Delete book from the list
    return NoContent();
}
