Chapter 6: Understanding Inheritance and Polymorphism

Introduction

In this chapter, we will learn about two important concepts in Object-Oriented Programming (OOP): inheritance and polymorphism. These concepts help us create related classes and allow objects to behave in different ways.

Inheritance

Inheritance allows us to create a new class based on an existing class. The new class (child) inherits properties and methods from the existing class (parent).

Example

// Parent class
class Car {
    public int MaxSpeed;
    public int Speed;
}

// Child class
class MiniVan : Car {
    // Inherits MaxSpeed and Speed from Car
}

Polymorphism

Polymorphism allows us to use a single interface to represent different underlying forms (data types). It means that a method can do different things based on the object it is acting upon.

Example

class Car {
    public virtual void Drive() {
        Console.WriteLine("Car is driving");
    }
}

class MiniVan : Car {
    public override void Drive() {
        Console.WriteLine("MiniVan is driving");
    }
}

Exercises

Exercise 1: Create a Simple Inheritance

  1. Create a base class Animal with a method MakeSound().
  2. Create a derived class Dog that inherits from Animal and overrides MakeSound() to print "Bark".

Pseudocode Hint:

class Animal:
    method MakeSound():
        print "Some sound"

class Dog inherits Animal:
    method MakeSound():
        print "Bark"

Exercise 2: Implement Polymorphism

  1. Create a base class Shape with a method Draw().
  2. Create derived classes Circle and Square that override Draw() to print "Drawing Circle" and "Drawing Square" respectively.
  3. Create a list of Shape objects and call Draw() on each.

Pseudocode Hint:

class Shape:
    method Draw():
        print "Drawing Shape"

class Circle inherits Shape:
    method Draw():
        print "Drawing Circle"

class Square inherits Shape:
    method Draw():
        print "Drawing Square"

list of Shape shapes = [new Circle(), new Square()]
for each shape in shapes:
    shape.Draw()

Summary

  • Inheritance allows a class to inherit properties and methods from another class.
  • Polymorphism allows methods to do different things based on the object they are acting upon.
  • Use the virtual keyword to allow a method to be overridden and the override keyword to override a method in a derived class.
