Chapter 3: Core C# Programming Constructs, Part 1

This chapter introduces key concepts in C# programming. You'll learn about:

  • Building a program's application object.
  • The Main() method and top-level statements.
  • Fundamental C# data types.
  • Data type conversion techniques.
  • The var keyword for implicit typing.
  • Control flow using loops and decision constructs.

Breaking Down a Simple C# Program

C# programs must have all logic within a type definition (like a class). Here's how to create a simple C# console application:

  1. Create a Solution and Project:

    Use Visual Studio or the command line to create a new solution and console application.

    dotnet new sln -n Chapter3_AllProjects
dotnet new console -lang c# -n SimpleCSharpApp -o .\SimpleCSharpApp -f net6.0
dotnet sln .\Chapter3_AllProjects.sln add .\SimpleCSharpApp
  2. Write the Main Method:

    The Main method is the entry point of the application.

    class Program {
    static void Main(string[] args) {
        Console.WriteLine("Hello, World!");
    }
}

Using Top-Level Statements

C# 9.0 introduced top-level statements, allowing you to write simpler code without the need for a class or Main method.

Console.WriteLine("Hello, World!");

Data Types and Variables

C# has several built-in data types, like int, string, and bool. You can declare variables using these types.

int myInt = 0;
string myString = "Hello";
bool myBool = true;

Implicit Typing with var

You can use var to let the compiler infer the type of a variable.

var myInt = 0; // inferred as int
var myString = "Hello"; // inferred as string

Control Flow: Loops and Decision Constructs

C# provides several ways to control the flow of your program, including loops (for, foreach, while, do-while) and decision constructs (if-else, switch).

Example of a for loop:

for (int i = 0; i < 5; i++) {
    Console.WriteLine(i);
}

Example of an if-else statement:

if (myInt > 0) {
    Console.WriteLine("Positive number");
} else {
    Console.WriteLine("Non-positive number");
}

Exercises

  1. Create a Simple Console Application:

    Write a program that prints "Hello, World!" to the console.

    // Pseudocode Hint:
Start
Print "Hello, World!"
End
  2. Using Variables:

    Declare and initialize variables of different types (int, string, bool). Print their values to the console.

    // Pseudocode Hint:
Start
Declare int variable and assign a value
Declare string variable and assign a value
Declare bool variable and assign a value
Print all variable values
End
  3. Control Flow with Loops:

    Write a for loop that prints numbers from 1 to 10.

    // Pseudocode Hint:
Start
For i from 1 to 10
    Print i
End For
End
  4. Decision Making:

    Write an if-else statement that checks if a number is positive, negative, or zero.

    // Pseudocode Hint:
Start
If number > 0
    Print "Positive"
Else If number < 0
    Print "Negative"
Else
    Print "Zero"
End If
End
  5. Using var Keyword:

    Declare variables using var and print their types.

    // Pseudocode Hint:
Start
Declare var variable and assign an int value
Declare var variable and assign a string value
Print types of both variables
End
