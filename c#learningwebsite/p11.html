Indexer Methods

What are Indexer Methods?

  • Indexer methods let you access elements in a custom class like an array.
  • Example: myCollection[0] to get the first item.

How to Use Indexer Methods

public class PersonCollection {
    private List people = new List();
    public Person this[int index] {
        get { return people[index]; }
        set { people[index] = value; }
    }
}

Exercise: Create a Custom Collection

  1. Create a class BookCollection.
  2. Add a list of Book objects.
  3. Implement an indexer to get and set books by index.

Pseudocode Hint:

class BookCollection:
    list books
    method get(index):
        return books[index]
    method set(index, value):
        books[index] = value

Operator Overloading

What is Operator Overloading?

  • Allows custom classes to use operators like +, -, etc.
  • Example: Adding two Point objects.

How to Overload Operators

public static Point operator +(Point p1, Point p2) {
    return new Point(p1.X + p2.X, p1.Y + p2.Y);
}

Exercise: Overload the + Operator for a Vector Class

  1. Create a Vector class with X and Y properties.
  2. Overload the + operator to add two vectors.

Pseudocode Hint:

class Vector:
    int X
    int Y
    method operator +(Vector v1, Vector v2):
        return new Vector(v1.X + v2.X, v1.Y + v2.Y)

Extension Methods

What are Extension Methods?

  • Add new methods to existing types without modifying them.
  • Example: Adding a Print method to List.

How to Create Extension Methods

public static class MyExtensions {
    public static void Print(this List list) {
        foreach (int i in list) {
            Console.WriteLine(i);
        }
    }
}

Exercise: Create an Extension Method for Strings

  1. Create a static class StringExtensions.
  2. Add a method IsPalindrome to check if a string is a palindrome.

Pseudocode Hint:

static class StringExtensions:
    method IsPalindrome(this string s):
        return s == reverse(s)

Anonymous Types

What are Anonymous Types?

  • Quick way to create a class with properties without defining it.
  • Example: var car = new { Make = "Toyota", Model = "Camry" };

How to Use Anonymous Types

var person = new { FirstName = "John", LastName = "Doe" };

Exercise: Create an Anonymous Type for a Product

  1. Create an anonymous type for a product with Name, Price, and Category.

Pseudocode Hint:

var product = new { Name = "Laptop", Price = 999.99, Category = "Electronics" }

Unsafe Code and Pointers

What is Unsafe Code?

  • Allows direct memory manipulation using pointers.
  • Example: int* p = &myInt;

How to Use Unsafe Code

unsafe {
    int* p = &myInt;
    *p = 10;
}

Exercise: Swap Two Integers Using Pointers

  1. Create an unsafe method to swap two integers using pointers.

Pseudocode Hint:

unsafe method Swap(int* a, int* b):
    int temp = *a
    *a = *b
    *b = temp
