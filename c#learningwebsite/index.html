Introducing C# Programming Language

C# is a powerful programming language used to create a wide range of applications, from games to enterprise software. Learning C# will help you understand fundamental programming concepts applicable to many languages.

Why Learn C#?

  • Versatility: C# can be used for various types of software development, including web, mobile, and game development.
  • Efficiency: C# provides powerful tools and libraries that make it easy to write efficient and high-performance code.
  • Foundation: Learning C# gives you a strong foundation in programming that can be applied to other languages.

Basic Structure of a C# Program


using System;

class Program
{
    static void Main()
    {
        // This is where your code goes
    }
}

This is the basic structure of a C# program. You'll start by including the necessary namespaces, defining a class, and writing your code inside the Main method.

Variables and Data Types

In C#, you use variables to store data. Here are some common data types:

  • int: Stores whole numbers.
  • float: Stores numbers with decimals.
  • char: Stores a single character.

int age = 25;
float height = 5.9f;
char initial = 'A';

Basic Operations

You can perform basic math operations in C#:

  • + for addition
  • - for subtraction
  • * for multiplication
  • / for division

int a = 10;
int b = 5;
int sum = a + b; // sum will be 15

Exercises

  1. Create a Simple Program: Write a program that prints "Hello, World!" on the screen.

    Hint: Use Console.WriteLine to display text.

    Pseudocode Hint:

    
Start
Include the System namespace
Define a class and the Main method
Use Console.WriteLine to print "Hello, World!"
End the Main method
  2. Math with Variables: Create a program that calculates the sum of two numbers.

    Hint: Use int variables to store the numbers, then add them.

    Pseudocode Hint:

    
Start
Declare two integer variables and assign them values
Add the two variables and store the result in a new variable
Print the result using Console.WriteLine
End the program
  3. Storing Information: Write a program that stores your age and height, then prints them.

    Hint: Use int for age and float for height.

    Pseudocode Hint:

    
Start
Declare an integer variable for age and a float variable for height
Assign your age and height to the variables
Print the values using Console.WriteLine
End the program
